# FindMe 日報作成スキル

毎日のビジネス数字の記録と振り返りを通じて、FindMe日報を作成してObsidianに保存するワークフロー。

## 定数

- 保存先: `/Users/sanonatsumi/Library/Mobile Documents/iCloud~md~obsidian/Documents/index/FindMe/`
- テンプレート参照: `/Users/sanonatsumi/Library/Mobile Documents/iCloud~md~obsidian/Documents/index/FindMe/2026-00-00.md`

---

## 実行順序（必ずこの順番で）

```
2️⃣ 数字のやり取り
  ↓
3️⃣ 全体の見直し & FindMe（売上分析）
  ↓
4️⃣ ミラーワーク
  ↓
1️⃣ ヒアリング（5つのエピソード）
  ↓
5️⃣ 明日のゴール
  ↓
Obsidianに保存
```

**鉄則：1ターンに1つの質問。まとめて聞かない。各ステップが完了してから次へ進む。**

---

## Step 1: 開始の声かけ

```
今日を振り返りますね。本日のビジネス数字から確認していきます。
```

## Step 2️⃣: 数字のやり取り

### 2-1: ビジネス数字をヒアリング

```
前日のデータと比較して、今日の数字を教えてください。増減があったか教えてください：

【3️⃣登録された"タイ"】
- リスト総数：
- 新規登録数：
- ユニークアクセス数：
（昨日との増減は？）

【2️⃣申し込まれ"タイ"】
- 申込数：
- ユニークアクセス数：
- アクセス数：
（昨日との増減は？）

【1️⃣お願いされ"タイ"】
新しい個別相談があれば教えてください
```

受け取ったら前日比を確認して簡単に合成する。

### 2-2: 媒体の進捗をヒアリング

```
ありがとうございます。次に、各媒体の進捗はどうですか？

Instagram / Threads / YouTube / Lステップ / WordPress / note / Facebook
```

活動のあった媒体のみ記録する。

---

## Step 3️⃣: 全体の見直し & FindMe

**1ステップずつ聞く（回答を受けてから次へ）**

### 3-1: マイプロデューサー到達レベル確認

```
📈マイプロデューサー到達レベルは？

　❾成約率が目標値に到達している
　❽個別相談が埋まってる
　❼個別相談が入ってる
　❺メインの施策を始めてる
　❹メインの施策を始めてない
　❸メインの施策の企画が決まっている
　❷メインの施策の企画が決まってない
　❶そもそも自分の企画ができてない
```

選ばれたレベルを記録する。

### 3-2: 今日の振り返りを1つずつヒアリング

以下の順番で1問ずつ聞く：

1. 「今日のGoodyは？（良かったこと）」
2. 「今日のBaddyは？（課題・反省点）」
3. 「ゴールとのギャップは？」
4. 「次のステップは？」

4つ集めたら簡潔にまとめる。

---

## Step 4️⃣: ミラーワーク

1ステップずつ聞く（回答を受けてから次のステップへ）：

```
では、ミラーワークで今日の体験を掘り下げていきます。

❶【事実エピソード】何をした？
```

全5ステップを順番に：
- ❶ 事実エピソード（何をした？）
- ❷ 効果エピソード（どんな結果を生んだ？）
- ❸ 原因ポイント（なぜそうなった？）
- ❹ 意味づけ（この行動は私に何を教えてくれた？）
- ❺ 改善プランニング（次はどこを変える？）

5ステップ完了後、学びを1段落でまとめる。

---

## Step 1️⃣: ヒアリング（5つのエピソード）

**重要な出力ルール：**
- ストーリー形式で書く（短くなくてOK）
- ラフで人間味のある文体
- 「でも」「だって」「難しい」「無理」は使わない

5つのテーマで順番にヒアリング（1つずつ聞く）：

1. **👟ハジマリの想い** — 今日の初期の想いや動機は？
2. **💧使命・試練・トラブル** — 直面した制約や課題は？どう工夫した？
3. **🎉師匠や自分との会話・新しい価値基準との出会い** — 得た新しい視点は？
4. **🌟没頭した考動** — 深く取り組んだことや突破は？
5. **🌈新たに掴んだ夢や目標** — 今日生まれた新しい目標は？

全5つ集めたら、**一つながりのストーリーとして描く**。

---

## Step 5️⃣: 明日のゴール

```
最後に、明日のゴールを設定します。

【テーマ】夢や目標は？
【本質】それを叶えるためになくてはならないものは？
【目的】今月のダンドリから、明日は何を達成する？
【火事】目的達成の障害は？
【施策】具体的にすることは？（4つ以上）
```

各項目の意味：
- **テーマ** = 夢や目標
- **本質** = テーマを叶えるために、なくてはならないもの
- **目的** = 今月の計画から、そのゴールを達成するためにその日何を達成するか
- **火事** = 目的達成の障害・問題
- **施策** = 火事を消し、目的を達成するために具体的にすること（4つ以上）
